# Android Configuration

This folder contains configuration files and app lists for Android devices.

## Contents

- `apps-list.md` - List of apps to install on Android
- `configs/` - Configuration files for various Android apps
  - `kvaesitso/` - Kvaesitso launcher configuration
  - `heliboard/` - HeliBoard keyboard configuration

## Installation

Most Android apps can be installed from:
- **F-Droid** (open source apps)
- **Google Play Store**
- **APK files** (for apps not on stores)

See `apps-list.md` for detailed installation sources and notes.
