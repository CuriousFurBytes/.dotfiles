# Kvaesitso Launcher Configuration

Kvaesitso is a search-focused Android launcher.

## Backup/Restore

Kvaesitso supports backup and restore of settings:

1. Open Kvaesitso settings
2. Go to **Backup & Restore**
3. **Export settings** to create a backup file
4. Save the backup file here as `kvaesitso-backup.json`

## Configuration Files

Place your Kvaesitso backup files here:
- `kvaesitso-backup.json` - Full settings backup

## Applying Configuration

1. Install Kvaesitso from F-Droid or GitHub
2. Open Kvaesitso
3. Go to Settings â†’ Backup & Restore
4. Choose **Import settings**
5. Select the backup file from this directory
