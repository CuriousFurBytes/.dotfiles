# HeliBoard Keyboard Configuration

HeliBoard is a privacy-friendly keyboard for Android (fork of OpenBoard).

## Backup/Restore

HeliBoard supports backup and restore of settings:

1. Open HeliBoard settings (long-press comma key or go to Android Settings → Languages & Input)
2. Go to **Advanced** → **Backup and restore**
3. **Export settings** to create a backup file
4. Save the backup file here

## Configuration Files

Place your HeliBoard configuration files here:
- `heliboard_settings.zip` - Full settings backup (includes dictionaries)
- `user_dictionary.txt` - Custom dictionary words

## Dictionaries

HeliBoard supports multiple language dictionaries. Download from:
- GitHub: https://github.com/Helium314/aosp-dictionaries

Common dictionaries to install:
- English (US/UK)
- Your additional languages

## Applying Configuration

1. Install HeliBoard from F-Droid or GitHub
2. Enable HeliBoard in Android Settings → Languages & Input
3. Set HeliBoard as default keyboard
4. Open HeliBoard settings → Advanced → Backup and restore
5. Choose **Import settings**
6. Select the backup file from this directory

## Recommended Settings

### Appearance
- **Theme**: Material Dark/Light or custom
- **Key borders**: Personal preference
- **Show hints**: Enable for special characters

### Typing
- **Auto-correction**: Modest or Aggressive
- **Show suggestions**: Always
- **Block offensive words**: Personal preference
- **Gesture typing**: Enable if desired

### Privacy
- **Send usage statistics**: Disable
- **Personalized suggestions**: Disable for max privacy
- **Clipboard history**: Configure based on preference
